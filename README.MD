### How to use

#### 1. install ->

```
npm i tdp-cli -g
```

#### 2. use ->
```
cd <dir>
tdp create <projectName> [itcode] [pwd] [appid]

like this:
tdp create demo // choose a app to create project

or:

tdp create demo -itcode liuyc14 -pwd 66666 -appId 898
// create project for the app(898)
```

### For developer

```
-> 1. npm install
-> 2. npm run build <Or> npm run dev
-> 3. npm link
-> 4. cd workspace
-> 5. tdp test

```
